# @Zhang2023 对视频监控系统的定义
视频监控系统(Public security video surveillance system)是指在个人或组织在无法直接观察或未处在当场环境时，通过该系 统观察视频监控设备拍摄的实时、直观的视频图像画面反映被监控对象行为。视频监控 系统主要包括视频监控采集前端摄像机、取电线路、传输网络、交换机、硬件服务器、 视频监控应用软件和显示系统等。

# @Zhang2023 公安视频监控系统建设项目的定义
公安视频监控系统建设项目是指在党委政府支持下、公安机关牵头建设的、与视频监控系统相关的项目，其目的是为保障公共安全和社会稳定，为社会和公民提供稳定的生活、学习、发展环境。2005 年，公安部提出开展建设监控系统的意见，各地公安机 关陆续逐步开展公安视频监控系统建设开展，主要由各地党委政府投资，通过项目立 项、财政评审、委托招标、签订合同、项目实施、验收、运行维护等一系列流程，完成 视频监控系统建设，为公安各警种部门、政府单位提供视频图像查询服务，在震慑犯 罪、打击犯罪、服务交通出行、化解矛盾纠纷、维护社会治安、服务民生等领域发挥重 要作用。

# @li2017 论述雪亮来源于天网，政策是平安建设

“天网工程” 最开始是在城市中推广，而随着平安中国建设的推进，如何在基础安防设施较为薄弱的农村，更好地保障人们的生产和生活安全，逐渐成为了社会治理工作的重心。

## 政策
2015年9月，国家发改委、中央综治办等九部委联合印发《关于加强公共安全视频监控建设联网应用的若干意见》，“雪亮工程”开始向全国推广。
其中提出的目标是，到2020年，我国基本实现“全域覆盖、全网共享、全时可用、全程可控”的公共安全视频监控建设联网应用。
一年后，国家发改委共同批准临沂市等48个城市成为全国首批公共安全视频监控建设联网应用工程示范城市，并获得中央补助资金。
2016年10月召开的全国社会治安综合治理创新工作会议部署全面开展“雪亮工程”建设。
2017年6月，全国“雪亮工程”建设推进会在临沂市召开。

# @Luo2020 雪亮原始数据储存在本地，云储存结构化视频
雪亮工程建设基本是按照“视频级联、本地 存储”的原则进行展开，之所以进行本地存储是 因为视频数据存储量过大，存储成本较高，但对 结构化之后的价值数据而言，所需的存储容量 低，平台就需要有实现半结构化解析和结构化解 析两个步骤:半结构化解析主要是实现原始视频 中运动目标的检测和提取，完成原始视频向运动 目标图片的转化;结构化解析主要是按照语义关系，采用时空分割、深度学习、特征提取等技术 方法，实现对监控视频和图片等半结构化信息的 进一步分析，解析后的结构可组织成供计算机和 人理解的文本信息。

# @blackmore2023 提出从Citizen-State Relations，社会想象的角度来理解监控中公民的态度

从近代早期的法国到现代的中国，技术官僚对公民的监控一直是中央国家机器的一个特征。国家 实践和社会法律史塑造了公民在监控背景下可获得的 "社会想象"，不仅影响了公民的实践 "剧目" ，还影响了公民的 "社会想象"。这些社会想象不仅影响并代表了公民关于国家的目的以及国家应如何行动的观念(Taylor 2004: 111)。这些社会想象既影响也代表了公民对国家的看法，即国家的目的是什么，国家是如何行动 的，应该如何行动。例如，人们发现，对政府监控的态度受到国家安全观(Bakir 等，2015 年)、 公民对政府机构的信任(Trüdinger 和 Steckermeier，2017 年)、对警察合法性的看法(Gurinskaya ，2020 年)以及对透明度的认知(Nam，2018 年)的影响。在最近的研究中，Trüdinger 和 Ziller (2022 年)发现，对政府应对威胁有效性的认知在决定公民是否支持通信监控方面具有重要意义 ，而 Degli Esposti、Ball 和 Dibb(2021 年)发现，对安全机构仁慈性的认知对于公民接受国家对 通信技术的监控至关重要。

Dencik 和 Cable(2017:764)使用 "**监控现实主义 surveillance realism** "一词，不仅表示 "理所当然 "理解监控的概念， 还表示人们 "同时对广泛的数据收集感到不安和担忧，同时国家机构和社交媒体平台等一系列行为 者积极地将监控做法正常化并为其辩护"。他们认为，这种情况代表了一种 "实用主义和逆来顺受 " 的意识，而不是同意(Dencik and Cable 2017: 777)，是由于对个人数据的使用缺乏了解、透明度 和控制而形成的。他们继续认为，这说明公民挑战现状的手段有限。Dencik 和 Cable(2017 年) 借鉴了 Taylor(2004 年)的 "社会想象 "概念，解释了社会和道德秩序是如何由理所当然的共同合 法性意识构成的，尤其是在国家角色方面，包括与监控有关的方面。泰勒(2004:23)认为，"社 会想象 "指的是 "人们想象其社会存在的方式，他们如何与他人融合在一起，他们与同伴之间如何 相处，通常会满足的期望，以及作为这些期望基础的更深层次的规范性概念和形象"。"社会想象 " 隐含地定义了什么是可能的:例如，对合法性的积极看法，其核心意味着正义是可能的;消极看法则暗示着践踏是可能的。

# 中国的国家利益高于隐私
对隐私的态度仍然与西方自由民主国家有根本的不同，更加强调集体利益和国家安全 而不是个人权利(Laidlaw，2015: 141)。

中国政府处理隐私的一个关键方面是将国家利益置于个人权利之上。在这种情况下， 隐私往往被视为国家安全、社会稳定和经济发展的次要关切，监控措施被认为是保护国家和公民的必要手段(Qiang，2011: 308)。
这种观点植根于中国共产党对集体利益 和社会和谐的强调，以及它的专制政治制度，它把维护权力和控制放在个人自由之上 (Laidlaw，2015: 143)。

中国隐私的法律
2017年，该国颁布了《网络安全法》，其中包括与个人信息的收集、存储和使 用有关的规定，以及对数据保护和违约通知的要求(张，2017:229)。
，在2020 年，中国出台了《个人信息保护法》(PIPL)，进一步加强了对个人数据的保护，并 对数据处理者和控制者施加了更严格的义务(Yeung，2020: 17)。 
它仍然受到国家强调 控制和监视的严重影响(Laidlaw，2015:145)。
尽管引入了新的数据保护法，但中 国当局继续从事广泛的监控行为，往往透明度和监督有限，引起了人们对这些法律保 护在实践中的有效性的担忧(Zhang，2017: 231)。

文化
传统的儒家价值观强调 社会和谐和集体利益而非个人权利(Laidlaw，2015: 147)。这种文化背景可能导致中 国公民更容易接受监视和国家对私人领域的干预，以及与西方同行相比，对隐私的期 望值较低(Yeung，2020: 19)。

# 中国监控的辩论：国家安全vs个人隐私
批评者认为，中国当局的监控方式侵犯了基本人权，如言论自由权和隐私 权，而支持者则断言，在快速发展的数字环境中，这些措施对于维护社会稳定和国家 安全是必要的(Qiang，2011: 302)。 

1. 国家有可能滥用权力，以及这些措施的实施 缺乏透明度和问责制(Zuboff，2019: 155)。
	批评者认为，中国无处不在的监控设备使 当局能够根据个人和团体的政治信仰、种族或社会地位，任意将其作为目标，导致公 民的恐惧文化和自我审查(MacKinnon，2012: 51)。
	此外，监控行为缺乏独立的监督 和司法审查，令人担心国家权力可能不受制约，民主价值可能受到侵蚀(Bennett, Raab, 2006: 283)。

2. 对个人隐私权的影响。中国当局 对个人数据的广泛收集、分析和储存，往往是在公民不知情或不同意的情况下进行的 ，这引起了人们对作为基本人权的隐私受到侵蚀以及这些数据可能被滥用的担忧( Lyon, 2018: 104)。此外，先进的监控技术，如面部识别和人工智能的使用，也引起了 人们对大规模监控的可能性和建立一个监视社会"，其中个人被不断监视和跟踪(Zuboff, 2019: 159)

另一方面，中国监控做法的支持者认为，面对各种内部和外部威胁，如恐怖主义、分 裂主义和网络犯罪，这些做法对于维护国家安全和社会稳定是必要的(Qiang, 2011: 306 )。
他们争辩说，中国政府的监控方法是合理的，因为需要保护国家和公民免受这些 威胁，所采用的措施与所涉及的风险相称(Creemers, 2017: 559)。 

结论：一些学者和政策制定者认为，在中国，关于监控的道德辩论不应该被 设定为国家安全和个人隐私之间的二元选择，而应该是一种复杂的平衡行为，需要仔 细考虑其中的权衡和后果(Bennett, Raab, 2006: 285)。

（一）前端摄像机建设  
根据分类建设原则，我市公共安全视频监控前端摄像机建设共分三类，具体分类如下：  
1.公共区域视频监控（一类）。中心广场、繁华街区、加油站、地下通道、停车场要在周边出入口、主要通道及重点场所、部位应安装视频监控摄像机；城市监察、市政工程建设和管理养护重点及难点部位应安装视频监控摄像机。  
2.公共安全重点区域视频监控（一类）。在治安情况复杂的城乡结合部，案件易发、多发的街道、区域（含水域）、重点单位等治安重点部位和公共安全敏感部位应安装视频监控摄像机。  
3.道路交通视频监控（一类）。在全市铁路、高速公路、国省干线公路、交通流量大的城区及县（乡）道路、重点旅游公路、水域主航道的重要路段（覆盖水面）和桥梁（含桥上路面和桥下水面）、互通立交、隧道、涵洞和事故多发地等重要部位应安装不同类型的视频监控摄像机。其中国家主干高速公路平均2公里、国省道路平均15公里建设一处视频监控。县（乡）道路、重点旅游公路、水域主航道的重要路段（覆盖水面）和大型桥梁、互通立交、隧道、涵洞等重要点位（覆盖水面）应建设全覆盖的视频监控摄像机。  
4.交通设施视频监控（一类）。在火车站、机场、港口、码头、渡口、地铁站、公交车站等公共交通站点，高速公路服务区、收费站，各类交通设施加油（气）站、充电站应全部安装视频监控摄像机；地铁、公交、客轮、客运车辆等公共交通工具上应安装视频监控摄像机。  
5.农村视频监控（一类）。在乡（镇）行政机关和村民委员会所在地、乡村主要交通出入口、学校、幼儿园、卫生所、商业网点、乡镇企业所在地应安装视频监控摄像机。  
6.边界视频监控（一类）。在高速公路、国省主干和重要支干道路、水域航道的省际、市际、县际出入口以及城区（中心城镇）的主要出入口应安装高清电子监控卡口。  
7.机关单位视频监控（二类）。具有独立场所办公的各级党政机关和事业单位要在停车场、出入口、大厅、主要通道、办事窗口及其他要害部位应安装视频监控摄像机。  
8.无线指挥视频监控（二类）。采取单兵、车载、临时架设等方式，在所有公共勤务的车辆上，构建集视频指挥、治安巡控、信息采集、现场取证、实时监督等功能于一体的无线指挥视频监控网。政法系统所有制式警用交通工具（含车、船、飞行器等）及路政、卫生、水务、燃气、电力、海事等部门的制式抢险救援交通工具（含车、船、飞行器等），应全部安装无线移动视频监控设备，实现图像信息实时上传。  
9.金融机构视频监控（二类）。全市各银行、证券、保险机构等金融网点要在营业柜台、自助网点、运钞车交接款通道、金库、客户区域及其他现金、有价证券、重要凭证、金银等贵重物品存储、交接部位应安装视频监控摄像机。

# @Zhang2023 H市疫情中监控困难，公安无法调度的例子
执行资源是公安视频监控项目建设的重要保障，包括财力资源、人力资源、信息资 源、沟通资源和激励机制。在访谈过程中，各被访人均提出资金保障问题，近一半被访 人提出专业人才缺乏、与市政部门沟通困难等问题。LZ 县提出公安视频监控项目需要 政府支持，尤其是资金支持;FX 区提出申请财政资金较为困难，申请周期太长影响项 目建设和落地;HS 区提出市政施工手续繁琐，部分破路、占用绿植等情况沟通申请手 续麻烦。由此可见，执行资源在公安视频监控项目建设中极为重要。

案例 3:2021 年 1 月，H 市公安局收到省公安厅《关于紧急接入疫情隔离点视频监控的通 知》，要求市局视频部门牵头，3 天内接入本市所有报备的疫情隔离点视频监控，公安机关遇到的问 题是隔离点底数不明，管理单位未接到通知不予配合。为尽快完成接入工作，市公安局联系卫健委 获取到全市拟启用和已启用共计 400 余个疫情隔离宾馆数量和名称，并按属地将视频监控接入工作 通知各分县局。因市疫情防控领导小组尚未下发文件，仅由市公安局视频部门牵头此项工作，协调 卫健部门和各隔离宾馆难度较大，3 天内根本无法完成，全市整体工作进展缓慢。10 天后，市疫情 防控领导小组办公室发文，要求各区县党委政府承担资金保障，卫健牵头、公安配合，并划分了工 作职责，要求接入已启动 196 个隔离点宾馆的“三区两通道”视频监控，在省厅政策下达 15 天后完 成视频监控接入工作，在 H 市疫情防控决策发挥了关键作用。
通过案例 3 可以看出财力资源、人力资源、沟通资源对政策执行执行的影响，正因 为市防疫办的文件解决了财力、人力、沟通资源，才使得疫情隔离点视频监控顺利接 入。此外，信息资源使执行主体可以掌握更多政策文件、技术要求、科技发展和业务人 脉，有利于政策的准确执行;激励机制符合执行主体的自身利益，能刺激执行主体的正 向工作意愿，提高项目建设成效。
